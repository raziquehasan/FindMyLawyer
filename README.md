# Find My Lawyer

Find My Lawyer is a web platform that helps users find verified lawyers based on their legal needs.

## Features
- Lawyer registration
- Lawyer verification system
- Client search for lawyers
- Secure authentication
- Document upload

## Tech Stack
Frontend: React  
Backend: Node.js + Express  
Database: Supabase

## Installation

Clone the repository

git clone https://github.com/nandinipawar123/find-my-lawyer.git

Install dependencies

cd backend
npm install

cd ../frontend
npm install

Run the project

npm start
